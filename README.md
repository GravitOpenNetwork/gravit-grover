# Gravit Grover Network (GGN)
[!TODO: Добавить бейджи]

**Distributed Epistemic Search Protocol**

GGN — это децентрализованный движок для поиска оптимальных гипотез.
Он сочетает геометрию Gower, вероятностное взвешивание и Grover-усиление для достижения глобального консенсуса.

## Быстрый старт
```bash
pip install -e .
make run-cluster  # Запускает 3 узла
```

### Принцип работы
* Gower Layer: Оценка близости гипотез.
* Amplification: Усиление сильных гипотез.
* Consensus: Обмен информацией между узлами.
